# chat.diffTool

Configure an external diff tool to preview code changes generated by the write tool, instead of using the built-in inline diff.

## Overview

When `chat.diffTool` is set, the CLI uses the specified diff tool to display code changes produced by the write tool.  
If the tool cannot be invoked successfully, the CLI falls back to the built-in inline diff.

## Usage

### Set Diff Tool

```bash
kiro-cli settings chat.diffTool <tool>
````

### Disable Diff Tool

```bash
kiro-cli settings -d chat.diffTool
```

Disables custom diff tools and restores the built-in inline diff.

## Value

**Type**: String
**Default**: *unset*

The value may be:

* A diff tool executable name (e.g. `delta`, `code`, `meld`)
* A command with arguments (e.g. `"delta --side-by-side"`)

## Examples

### Example 1: Terminal Diff Tool

```bash
kiro-cli settings chat.diffTool delta
```

Uses `delta` to render diffs in the terminal.

### Example 2: GUI Diff Tool

```bash
kiro-cli settings chat.diffTool code
```

Opens diffs in VS Code.

### Example 3: Custom Arguments

```bash
kiro-cli settings chat.diffTool "delta --side-by-side"
```

Uses `delta` with side-by-side layout.
