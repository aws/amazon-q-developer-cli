Set up SSH Integrationbeta

Fig will make changes to your SSH config (stored in ~/.ssh/config).

How does it work?

Fig prepends the following text to the start of ~/.ssh/config using the Include directive. This sends Fig a signal whenever a new SSH session begins and allows Fig to use the same connection. 

    ControlPath ~/.ssh/%C
    ControlMaster auto
    ControlPersist 1
    PermitLocalCommand yes
    LocalCommand fig bg:ssh ~/.ssh/%C &

You can disable this at any time by editing your ~/.ssh/config file and removing the integration.
