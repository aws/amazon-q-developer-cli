

##### Install protobuf compiler:

```
 brew install protobuf
```



##### Install Swift compiler

```
brew install swift-protobuf
```



##### Install TS compiler

```
npm install -g ts-proto
```

##### Install go compiler
```
go install google.golang.org/protobuf/cmd/protoc-gen-go@latest

```


##### Compile protos

```
./proto.sh
```
